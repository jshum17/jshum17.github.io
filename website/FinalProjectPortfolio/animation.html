Animation

Flash Cartoon

The Flash Cartoon assignment was one of my favourite assignment. I created the animation using Adobe animate. This animation is a result of the graphic design and animating skills that I learned during this class. This assignment has also pushed myself to become more creative.

Frame Animation

The Frame Animation assignment was an interesting assignment that went through a lot of process. The mix of a paper template, Photoshop and Adobe animate were involved in the making of this animation. By importing and aligning each image box to different frames, I was able to create this cool frame-by-frame animation.

Bouncing Ball

The Bouncing Ball assignment was another frame animation assignment that I created using Adobe Animate. Instead of using a paper template to draw out the story board, I created each frame entirely on Adobe Animate. I am very proud of how the resulting animation turn out to be.

Animate Cartoon

The Animate Cartoon assignment was an edit of the dexter animation that was given to us. This assignment was a good test of our knowledge regarding movie clips and fill objects. I created this cartoon using Adobe Animate and I enjoyed working it a lot.

Colour Book

The Colour Book assignment was another fun project that I really enjoyed working on. I created the colour book using Adobe Animate. I drew out the characters and added actionscripts to make it interactive.
